# MC CPM Computer Clone
 
Hier entsteht ein MC CP/M Computer kompatibler Rechner. Dieser wurde in der Zeitschrift MC Computer 1984 als NAchbauprojekt beschrieben. Der Author war Rolf-Dieter-Klein der auch den NDR Computer gleichzeitig entwickelte.
Das Design sollte weitgehend mit Chips aus der Zeit aufgebaut werden, also kommen keine SMD Bauteile zum Einsatz.
